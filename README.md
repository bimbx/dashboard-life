# Productivity Dashboard

A comprehensive productivity dashboard with task management, habit tracking, and health monitoring.

## Features

- Task Management
- Pomodoro Timer
- Habit Tracking
- Calendar
- Quick Notes
- Work Mode
- Position Tracking
- Mood Tracking
- And more!

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- NextAuth.js
- Prisma
- PostgreSQL

## Getting Started

### Prerequisites

- Node.js 18+
- PostgreSQL database
